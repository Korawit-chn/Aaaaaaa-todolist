# Food Delivery System

## Overview
The Food Delivery System is an online platform that allows customers to browse menus, place food orders, and pay electronically. Restaurants receive orders and update preparation status, while administrators manage the system.

## Users
- Customers: order food
- Restaurant Staff: manage orders
- Admin: configure system

## External Systems
- Payment Gateway for online transactions

## Containers
- Web App (React)
- Mobile App (Flutter)
- Backend API (Node.js/Express)
- Database (PostgreSQL)
- Payment Service

## Components (Backend API)
- Auth Controller
- Order Service
- Menu Service
- Repository Layer
